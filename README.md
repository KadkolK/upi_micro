# upi_micro

# UPI Micro-Fraud Detection System

A multi-phase project to simulate and detect micro-fraud in UPI transactions using Machine Learning and real-time systems.

---

## Project Structure

* **Phase 1** → Synthetic Data + ML Model
* **Phase 2** → Fraud Detection API
* **Phase 3** → Real-time Transaction Simulation
* **Phase 4** → Alert & Logging System
* **Phase 5** → Dashboard & Monitoring

---

## Objective

Detect suspicious UPI transactions (₹1500–₹2000 range) using:

* Behavioral patterns
* Transaction velocity
* Device & location anomalies
